The Marble Language Tool is created by MarbleFlower and FerdysLab9 for use in Marble's commissioned artwork and fictional settings. There is one modern alphabet available for now, but more writing systems are planned for the future. The alphabet is generally phonetic, and what is written is typically what is actually pronounced. The alphabet can be used to write many different real world and fictional languages, so some letters are assigned to multiple sounds. These sounds are allophones, enclosed in square brackets, and would be understood as the same sound in the language that Marble and other characters speak.

The language can be transcribed with a Romanisation system for readability. The corresponding transcription for each letter is enclosed in angle brackets. All letters can be typed with a standard QWERTY keyboard without diacritics, with the exception of an optional diaeresis for syllable pacing (for example <Zoë>). The letters C, J, Q, and X are not used for unambiguity.
